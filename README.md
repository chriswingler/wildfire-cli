# Wildfire CLI

A text-based wildfire incident commander simulation game that teaches real-world firefighting tactics and decision-making.

## Overview

This game simulates the role of a wildfire Incident Commander using authentic wildfire management principles. Players make strategic decisions about resource deployment, tactics, and risk management while managing realistic fire scenarios.

## Features

- **Realistic Incident Command System (ICS)** - Based on actual wildfire management protocols
- **Text-based Interface** - Professional terminal UI using Rich library
- **Multiple Scenarios** - Progressive difficulty from grass fires to complex multi-agency incidents
- **Educational Content** - Learn real firefighting terminology and tactics
- **Resource Management** - Deploy ground crews, helitack, air tankers, and bulldozers strategically

## Installation

```bash
pip install -r requirements.txt
python src/main.py
```
